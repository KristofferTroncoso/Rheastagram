## frontend
* React
* Ant-Design components
* AWS Amplify components
* css flexbox
* graphql api


## backend
* AWS Amplify
* AppSync (graphql)
* DynamoDB (NoSQL Database)
* S3 (Storage)
* Cognito (Auth)
